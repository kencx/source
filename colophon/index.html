### Software

The site is built with Hugo and hosted on a Hetzner server behind Caddy. The full source code is available on Github and DNS is hosted on Cloudflare.

The theme is adapted from Bamboo CSS and the font is IBM Plex Mono. All icons are obtained from feathericons.

I write my notes using a combination of neovim and Obsidian.md. You can find my dotfiles here.
