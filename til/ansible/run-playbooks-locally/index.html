Run Playbooks Locally
3 Jan 2023 • 1 min

## From the command line

$ ansible-playbook --connection=local 127.0.0.1, main.yml
Note: The comma is mandatory after 127.0.0.1,, otherwise 127.0.0.1 is treated as a filename.

## With an inventory file

[local]
localhost ansible_connection=local

$ ansible-playbook -i inventory --limit local main.yml

## Within a playbook

### All tasks

- name: Playbook example
  hosts: 127.0.0.1
  connection: local
  tasks:
    ...

### A single task

Using the local_action module:

tasks:
  - name: Local task
    local_action:
      module: git
      repo: ...
      dest: ...

or using the delegate_to keyword:

tasks:
  - name: Local task
    git:
      repo: ...
      dest: ...
    delegate_to: localhost
