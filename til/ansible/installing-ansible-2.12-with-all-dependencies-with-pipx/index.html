Installing Ansible 2.12 With All Dependencies With Pipx
29 Dec 2022 • 1 min

TLDR: See here for the full script.

Ansible’s release history is really confusing. Ansible’s core package has taken on a number of names:

  • ansible (pre 2.9)
  • ansible-base (2.10)
  • ansible-core (post 2.10)

and from Ansible 2.10, Ansible community packages have also split off into their own “Ansible Community package” which has a different release cycle and its own versioning system.

To properly install the latest version of Ansible 2.12 without dependency issues, I use pipx

# install or update pipx
$ python3 -m pip install --user pipx
$ python3 -m pipx ensurepath

# install ansible
$ pipx install --include-deps ansible

To include ansible-lint

$ pipx inject --include-apps ansible ansible-lint

To include molecule with the Docker and Vagrant driver,

$ pipx inject --include-apps ansible molecule
$ pipx inject --include-deps --include-apps ansible molecule-docker
$ pipx inject --include-deps --include-apps ansible molecule-vagrant
