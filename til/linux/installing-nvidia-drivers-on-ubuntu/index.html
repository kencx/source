Installing Nvidia Drivers on Ubuntu
14 Apr 2024 • 1 min

This guide is for Ubuntu/Debian systems only.

## Installation

  1. Identify your graphics card
$ lshw -C display

  1. Determine the necessary driver version for your card at the Nvidia site.

  2. Search for a list of Nvidia drivers in Ubuntu/Debian systems with

$ apt search nvidia-driver | less
  1. Install the appropriate driver recommended by Nvidia in step 2.
$ sudo apt install nvidia-driver-510
  1. Reboot your system.

## Updates

There are 2 forms of driver updates - minor and major.

#### Minor Updates

Minor updates happen when there is a patch to the currently installed driver version. Nvidia drivers do not update with sudo apt upgrade, but with sudo apt full-upgrade instead.

  1. Check the update list to ensure you know what you are updating
$ sudo apt list --upgradable
  1. Perform the update
$ sudo apt full-upgrade
  1. Reboot your system.

#### Major Updates

Major updates happen when there is a new driver released.

  1. Firstly, check that the driver is compatible with your graphics card on the Nvidia site. Older cards might not be supported.

  2. Perform a full installation of the new version with sudo apt install. The installation should remove any old driver versions at the same time.

$ sudo apt install nvidia-driver-[new-version-num]
  1. Reboot your system
