Editing Archive Files With Vim
14 Apr 2024 • 1 min

Vim can be used to edit archive files of multiple formats directly1:

$ vim foobar.zip
$ vim foobar.tar.gz

## References

:help tar
:help zip
:help gzip

  1. Interestingly, I discovered this when I accidentally opened an .epub file with vim ↩︎
